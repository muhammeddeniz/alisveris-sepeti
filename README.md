### Gerekli Paketler

* [React](https://reactjs.org/)
* [Mobx](https://mobx.js.org/README.html)

### Kurulum
 
1. Projeyi Klonlayın
```sh
git clone https://github.com/muhammeddeniz/react-starterkit.git
```
2. Yarn Paketlerini İndirin
```sh
yarn
```

## Geliştirme 

```sh
yarn
yarn start
```

http://localhost:3000/ adresinden projeye ulaşabilirsiniz
